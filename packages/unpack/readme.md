# rail-style/unpack

A set of helper functions which reverse packings performed by the
[pack](https://www.npmjs.com/package/@rail-style/pack) module.

| Function | Min. Length | Max. Length |
|----------|-------------|-------------|
| tiploc   | 1           | 7           |
| crs      | 3           | 3           |
| trainUid | 6           | 6           |
